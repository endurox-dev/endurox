BBOOLPR(3)
==========
:doctype: manpage


NAME
----
Bboolpr - Print the compiler boolean expression AST


SYNOPSIS
--------

#include <ubf.h>

void Bboolpr (char *'tree', FILE *'outf');

Link with '-lubf -lnstd -lm -lpthread'

DESCRIPTION
-----------
Print the compiled UBF buffer boolean expression evaluation Abstract Syntax Tree
 (AST) at 'tree' hander to 'outf' stream. The sample output for AST is following:

--------------------------------------------------------------------------------

...
tree=Bboolco ("2 * ( 4 + 5 ) || 5 && 'abc' %% '..b' && 2/2*4==5")
...
Bboolpr(tree, stdout);

will give:
((2*(4+5)) || ((5 && ('abc' %% '..b')) && (((2/2)*4) == 5))

--------------------------------------------------------------------------------


RETURN VALUE
------------
Function is void and does not return any value.

ERRORS
------
Function is void and does not return any error.
